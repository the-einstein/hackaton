Registration API

post request to https://api-data-chellenge.herokuapp.com/user/register

        
            {
                name: "name minimum 5 characters",
                phone: "minimum 9 charackters",
                password: "minimum 8 charackters",
                passport: "passport serial number, minimum 9 charackters"
            }

If error returns error massage

        
            "name" length must be at least 5 characters long

Login API (on development, only returns user)

post request to https://api-data-chellenge.herokuapp.com/user/login

        
            {
                phone: "999643690",
                password: "password"
            }

or

        
            {
                passport: "AB5466773",
                password: "password"
            }

Application form API

post reques to https://api-data-chellenge.herokuapp.com/app/new

        
            {
                title: "required: title of form (corruptin, sanitary, ...)"
                details: "required: text of application",
                to: "required: Organ id to which application form is sent",
                from: "required: Application user id",
                score: "required: score given for organ",
                proofs: "file or files",
                anonymous: "boolean"
            }

Getting all organs list

get request to https://api-data-chellenge.herokuapp.com/organ

returns array of objexts

Getting Region list

get request to https://api-data-chellenge.herokuapp.com/organ/regions

returns only Tashkent city
